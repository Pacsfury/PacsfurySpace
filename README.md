# PacsfurySpace
---

## Approach
I'm currently working on this project, which consists on creating a suite of fast, minimalist, useful, secure, private (and more) webs.

All the webs are adapted to my needs, and I want it to be adapted to yours too. 

This is in early development, so I only have a web, but expect more coming soon.

## Current Webs

### - Documents
The most basic workspace app, isn't it? 

Write text with some new functionalities.

#### Content

- **Format**: Bold, Italic, Underline
- **Personalize**: Color, Font for your texts
- **Organize**: Every document has a own tree structure with children and parents, just to keep yourself organized.
- **Write**: Write texts, basic feature
