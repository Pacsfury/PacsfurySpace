# PacsfurySpace
<sub>**version:** _alpha 0.0.1_</sub>

> A private and simple suite for your everyday use
---

## Approach
I'm currently working on this project, which consists of creating a suite of fast, minimalist, useful, secure, and private web applications.

All the webs are adapted to my needs, and I want them to be adapted to yours too. 

This is in early development, so there is only one web app available, but expect more coming soon.

I just want to make things clear: this is _private_, _fast_, and _lightweight_ by design. No servers, no telemetry, no bloatware. Just you and your computer.

To make it even better, it is open-source so everyone can contribute to making it theirs and better.

## Technology
As I said, this is meant to be fast, so we are using mostly vanilla:
- HTML5 vanilla
- CSS3 vanilla
- JS vanilla
- IndexedDB

## Current Webs

### Documents
The most basic workspace app, a text processor. 

Write text with some new functionalities.

#### Features
- **Format**: Bold, Italic, Underline
- **Personalize**: Color, Font for your texts
- **Organize**: Every document file has its own tree structure with children and parent documents, just to keep yourself organized and neat.
- **Write**: Distraction-free text writing.

---

## How to Run Locally
Since this project uses no build tools or heavy frameworks, running it is simple:
1. Clone this repository.
2. Open the `index.html` file of the app you want to use directly in any modern browser.
3. That's it! No `npm install`, no setup required.
